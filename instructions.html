Nano Grid Instructions

How to Play

The object of the same is to find all of the hidden blocks.

The numbers above and before the columns and rows represent how many continuous blocks are in each. An example solution is shown below.

Simply click any box to fill it. Click it again to set a red X. These help to remind you which blocks cannot be filled.

Right clicking a box will start measurement line. Simply move the mouse and the block count from the clicked box will appear. Use this to help you counting.

The game will automatically tell you when you have found a successful solution. 

Menus

File

Load/Save Game -  Load and save the current game, including marks.

Load/Save Puzzle -  Load and save the current puzzle excluding marks.

Settings

Grid Options - Opens a dialog which allows for custom grid creation options. Clicking OK from this dialog will regenerate the puzzle.

Columns/Rows: the grid size.

Max Columns/Row - The maximum number of squares in each column or row. Keep in mind that after actual generation the combination of column and row marks may lead to more marks than these set maximums. Usually the higher this number the easier the game will be. 

Break % - The percent chance that a row will be split during the grid generation. Typically higher numbers will result in scattered puuzzles. A 0% is the standard value.

Refresh Puzzle - Clears all marks from the existing puzzle.

New Puzzle - Generate a new puzzle using the current settings.

Hint

Check - Display the number of misplaced marks.

Peek - Display the puzzle solution for 1 second.

Show - Show the completed puzzle.
